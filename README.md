# Mushee Yield — Powered by Starkzap

**Mushee (UK incorporated)**  
Tagline: **Make yield feel like a button.**

Mushee Yield is a complete Starkzap demo built around a simple loop:

**connect → get testnet funds → load pools → stake → track → claim**

## Included in this build
- Cartridge wallet onboarding via Starkzap
- Validator presets and pool discovery
- Pool metrics cards
- Faucet section for Starknet Sepolia funding
- Bitcoin Birthday Cards prototype
- Spin the Wheel reward demo surface
- Ready Wallet section
- Links section for demo and repo URLs

## Run locally
```bash
npm install
npm run dev
```
